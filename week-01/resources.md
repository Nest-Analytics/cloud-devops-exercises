## SSH basics

- SSH command format refresher:
  - `ssh -i <private_key_path> <user>@<public_ip>`
- Reminder:
  - Do not share your private key file.
  - If you paste terminal output into notes, redact IP addresses if requested by the trainer.

## Git basics (for cloning the demo app)

- Install Git on Ubuntu:
  - `sudo apt update && sudo apt install -y git`
- Clone:
  - `git clone <repo_url>`
