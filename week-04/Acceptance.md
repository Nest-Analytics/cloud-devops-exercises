# Acceptance Criteria

A  submission is marked complete when all items below are met.

## Submission structure
- Work is submitted in `cloud-devops-submissions` under `week-04/`.
- `notes.md` is included.
- `screenshots/` folder is included.

## Web App running
- A working Web App URL is provided in `notes.md`.
- The Demo App loads successfully on the student Web App URL.

## ZIP deploy completed
- `notes.md` confirms ZIP deploy was done successfully.
- Screenshot evidence shows the app running after ZIP deploy (live page).

## Environment variable configuration (`APP_TITLE`)
- Screenshot evidence shows all the env present in Web App App settings.

## GitHub CI/CD deployment completed
- Screenshot evidence shows GitHub deployment is configured (repo and branch visible).
- Screenshot evidence shows a GitHub-triggered deployment occurring (log/status).
- Screenshot evidence shows the deployed app reflects a change pushed to GitHub.
- `notes.md` describes the change used to confirm CI/CD is working.
