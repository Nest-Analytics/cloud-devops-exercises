
## Resources

Use these resources to support the Week 2 assessment (processes/logs, PM2, and Git workflow). Do not try to read everything. Use them when you get stuck or when you want deeper understanding.

---

### PM2 (Process Manager)

**PM2 Quick Start**  
Use this to understand the basic PM2 commands used in this week (`pm2 start`, `pm2 list`, `pm2 logs`, `pm2 save`, `pm2 startup`).  
https://pm2.keymetrics.io/docs/usage/quick-start/

**PM2 Process Management**  
Use this when you want to understand how PM2 manages processes, restarts apps, and how to view process details and logs properly.  
https://pm2.keymetrics.io/docs/usage/process-management/

**PM2 Troubleshooting Handbook (extra reading)**  
Use this if your app does not start, PM2 shows errors, logs are confusing, or you want a structured approach to troubleshooting (PM2, Nginx, MySQL, SSH, Linux).  
https://medium.com/@lahin31/complete-troubleshooting-handbook-for-pm2-nginx-mysql-ssh-and-linux-4e8df27fd9c0

---

### Git (Official Docs)

**Git Tutorial (official)**  
Use this to learn the standard Git workflow: cloning, branching, staging, committing, pushing, and understanding history with `git log`.  
https://git-scm.com/docs/gittutorial
